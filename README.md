### 功能说明

* 满足搜索到某个固定的有名号的资源（非老司机无法使用）
* 仅仅提供公用资源100页（如果有老司机愿意捐献账号，可以爬更多）
* 可满足速度调节（默认一分钟，请留一点资源贡献给所有人）

### 运行方式

CGI运行
``` php
php 1024.php
```

如果需要网站方式运行，请自行加入网站超时函数
